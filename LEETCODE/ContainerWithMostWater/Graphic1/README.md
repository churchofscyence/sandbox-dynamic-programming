## Graph Layout with Scaling Factor

Scale x (Sx) by 40       
Scale y (Sy) by 40   

800px (Width of the canvas) = 20 * Sx     
600px (Height of the canvas) = 15 * Sy     
80px (Width between the Vertical Lines) = 2 * Sx
